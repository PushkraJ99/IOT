
#### RFID Pins Connection

| Parameter | Pin Number     |
| :-------- | :------------- |
| `3.3 V`   |    **PIN 1**   |
| `RST`     |   **PIN 22**   |
| `GND`     |   **PIN 6**    |
| `MISO`    |   **PIN 21**   |
| `MOSI`    |   **PIN 19**   |
| `SCK`     |   **PIN 23**   |
| `SDA`     |   **PIN 24**   |


## Open Terminal & Type Below Commands

 Run Python Program

```bash
  cd LED 
  python rfid.py
```
OR
```bash
  cd LED
  python3 rfid.py
```
